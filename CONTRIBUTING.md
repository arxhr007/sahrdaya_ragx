# Contributing

Thank you for your interest in improving this project.

## Allowed Contributions

Contributions are allowed only through this repository's official workflows (issues, pull requests, and maintainer-approved changes).

## Ownership and License Terms

By submitting a contribution, you agree that:

- You have the right to submit the contribution.
- Your contribution may be modified, merged, or rejected by maintainers.
- Your contribution becomes part of this repository under the repository's proprietary terms.
- You do not receive rights to reuse, redistribute, sublicense, or sell the full codebase or derivative copies outside this repository unless explicitly permitted in writing by the copyright holder.

## Pull Request Guidelines

- Keep pull requests focused and small.
- Clearly describe what changed and why.
- Link related issues when relevant.
- Do not include secrets, API keys, or credentials.
- Respect existing code style and file structure.

## Prohibited Actions

- Republishing this codebase (or substantial parts) as your own work.
- Commercial use or resale of this codebase without written permission.
- Relicensing any part of this project without maintainer approval.

## Enforcement

Maintainers may close pull requests or revoke collaboration access for policy violations.
